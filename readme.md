# Music Playlist Manager (Multi-Playlist)
# MAde on python version 3.13.7
Modern GUI (ttkbootstrap) music playlist manager with multiple playlists, doubly linked lists per playlist, and SQLite persistence.

## Run

1. Install dependencies:
pip install -r requirements.txt
2. Place audio files in `songs/`.
3. Start the app:

4. Create playlists from the sidebar (e.g., "favourite Songs", "90s Hits"), import songs, and manage each list separately.

## Features
- Multiple playlists (create, rename, delete, switch)
- Add, delete, search, play songs per playlist
- Doubly linked list structure per playlist
- Full-screen modern GUI (ttkbootstrap themes)
- Persistent storage across sessions (SQLite)
